This is a _wrapper_ to amuellers wordcloud generator for specific usage
for WhatsApp conversations.

The script uses a config file where you can specify words to ignore, images to use, paths etc.
 
 
USAGE:
 * Install word_cloud from https://github.com/amueller/word_cloud: 
    pip install wordcloud
 * Adjust config.ini to your own specifications
 * Run wordcloud_whatsapp.py
